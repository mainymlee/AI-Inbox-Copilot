# spec.md — Newsletter Manager

## 목표

> 뉴스레터 및 광고성 이메일을 분석하여 발신자별로 그룹화하고, 사용자의 열람 패턴을 기반으로 효율적인 관리 방법을 추천한다.

---

## 맥락

### 대상

* 뉴스레터와 광고 메일이 많이 쌓이는 사용자
* 메일함 정리가 어려운 사용자

### 사용 목적

반복적으로 수신되는 뉴스레터를 효율적으로 관리하고,
중요하지 않은 메일로 인해 중요한 메일을 놓치지 않도록 돕는다.

---

## 범위

### 입력

* Mail Summary 결과
* Mail Classification 결과
* 읽음 여부
* 수신 날짜
* 발신자
* 뉴스레터 및 광고 메일 목록

### 출력

Markdown 형식의 뉴스레터 관리 결과

포함 내용

* 뉴스레터 현황
* 발신자별 그룹
* 장기간 읽지 않은 뉴스레터
* 추천 관리 방법
* 주간 요약 추천

---

## 제약

* 메일을 삭제하거나 이동하지 않는다.
* 구독을 해지하지 않는다.
* 구독 해지는 추천만 한다.
* 발신자 기준으로 그룹화한다.
* Markdown 형식을 유지한다.
- 모든 출력은 공통 출력 형식의 "다음 Skill 전달 정보" 섹션을 포함한다.
- 다음 Skill이 사용할 수 있도록 핵심값을 명확한 항목명으로 작성한다.
- 없는 정보는 추측하지 않고 "없음" 또는 "확인 필요"로 표시한다.
---

## 출력 형식

파일명

NewsletterManager.md

섹션 순서

1. 뉴스레터 현황
2. 발신자별 그룹
3. 장기간 읽지 않은 뉴스레터
4. 추천 관리 방법
5. 주간 요약 제안

분량

300~600자

---

## 성공 기준

* [ ] 뉴스레터를 발신자별로 그룹화한다.
* [ ] 장기간 읽지 않은 뉴스레터를 식별한다.
* [ ] 추천 관리 방법을 제공한다.
* [ ] 구독 해지는 추천만 한다.
* [ ] Safe Archive에서 사용할 수 있는 결과를 생성한다.
